安装篇

我安装的是目前最新版本Python 3.7.3，传送门，选择 Download Windows x86 executable installer大概25M左右。
下载好之后就运行文件，需要勾选Add Python 3.7 to PATH，即配置环境变量，忘记勾选的话就需要手动去配置环境变量了，否则，从新安装即可。然后点“Install Now”即可完成安装。
我建议在windows装个 git，不然后面用windows的命令提示符操作好变扭，反正我是用不惯。
随便创建一个学习python用的文件夹 D:\python，在这个文件夹下面运行 Git Bash，然后运行 python –version 如果能显示python 版本号，即安装成功。
